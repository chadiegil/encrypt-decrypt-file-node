To encrypt the file  type the command = node encrypt.js

To decrypt  the file  type the command = node dencrypt.js


=================================
|           NOTE                |
=================================

When the encrypt file is run, it will create an output file called output.txt.enc that is in binary format. When the decrypt file is run, it will take the output.txt.enc file and decrypt it back to its original, unencrypted text content.
